Módulo 1: Núcleo y Proyectos

La base fundamental del sistema Nexus Obra.

Ver Tablas de Base de Datos

Objetivo del Módulo

Establecer la estructura base del software permitiendo la autenticación segura de los usuarios y la creación de la entidad "Proyecto" (Obra).

Sin este módulo, no existen los ID de proyecto necesarios para ligar contratos, estimaciones o reportes de avance. Es el "padre" de toda la información.

Funcionalidad por Tipo de Usuario

El Administrador

Control Total

Es el encargado de configurar el sistema.

  • Crear Obras: Dar de alta nuevos proyectos.
    Inputs: Nombre, Centro Costos, Fechas.
  • Gestionar Usuarios: Crear cuentas y asignar roles.

El Director

Solo Lectura

Visión global de la empresa.

  • Consultar Listado: Ver todas las obras activas y su estatus general.
  • Ver Personal: Consultar quién tiene acceso al sistema.

El Residente

Usuario Operativo

Acceso operativo.

  • Login: Iniciar sesión en la App Móvil.
  • Contexto: El sistema lo asigna automáticamente a su obra.
