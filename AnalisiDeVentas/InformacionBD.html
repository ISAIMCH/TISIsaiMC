Documentación de la Base de Datos de Análisis de Ventas

Módulo: Análisis de Ventas

Diagrama de la Base de Datos

Tabla: Analisis

Propósito: Almacenar información general del análisis de ventas en un período determinado, incluyendo total de ventas, ticket promedio y utilidad neta.

Campo Tipo de Dato Permite Nulo Propósito / Justificación
ID_Analisis INT No Clave primaria que identifica de manera única cada análisis.
FechaInicio DATE No Fecha de inicio del período del análisis.
FechaFin DATE No Fecha final del período del análisis.
TotalVentas DECIMAL(10,2) No Total de ventas generadas en el período.
TicketPromedio DECIMAL(10,2) No Promedio de venta por ticket.
UtilidadNeta DECIMAL(10,2) No Ganancia neta obtenida en el período.
FechaRegistro DATETIME No Fecha y hora de registro del análisis.

Tabla: ClientesFrecuentes

Propósito: Registrar clientes frecuentes y su comportamiento de compra dentro de un análisis específico.

Campo Tipo de Dato Permite Nulo Propósito / Justificación
ID_ClienteFrecuente INT No Clave primaria que identifica cada registro de cliente frecuente.
ID_Analisis INT No Llave foránea hacia la tabla Analisis, relaciona al cliente con el análisis.
ID_Cliente INT No Identificador del cliente.
ComprasRealizadas INT No Total de compras realizadas por el cliente en el período.

Tabla: ProductosMasVendidos

Propósito: Registrar los productos más vendidos durante un análisis específico.

Campo Tipo de Dato Permite Nulo Propósito / Justificación
ID_ProductoMasVendido INT No Clave primaria de la tabla.
ID_Analisis INT No Referencia al análisis correspondiente.
ID_Producto INT No Identificador del producto.
CantidadVendida INT No Total de unidades vendidas del producto en el período.

Tabla: HorariosPico

Propósito: Identificar los intervalos de tiempo con mayor número de ventas en un análisis.

Campo Tipo de Dato Permite Nulo Propósito / Justificación
ID_HorarioPico INT No Clave primaria de la tabla.
ID_Analisis INT No Referencia al análisis correspondiente.
Intervalo VARCHAR(50) No Intervalo horario con más ventas.
CantidadVentas INT No Número de ventas en el intervalo.

Tabla: DiasConMasVentas

Propósito: Registrar los días con mayor volumen de ventas dentro de un análisis.

Campo Tipo de Dato Permite Nulo Propósito / Justificación
ID_DiaMasVenta INT No Clave primaria de la tabla.
ID_Analisis INT No Referencia al análisis correspondiente.
Fecha DATE No Día con más ventas.
TotalVentas DECIMAL(10,2) No Total de ventas generadas ese día.

Tabla: MesesConMasVentas

Propósito: Identificar los meses con mayor volumen de ventas en un análisis.

Campo Tipo de Dato Permite Nulo Propósito / Justificación
ID_MesMasVenta INT No Clave primaria de la tabla.
ID_Analisis INT No Referencia al análisis correspondiente.
Año INT No Año del mes registrado.
Mes INT No Mes con más ventas.
TotalVentas DECIMAL(10,2) No Total de ventas del mes.

Tabla: EmpleadosMasVentas

Propósito: Registrar empleados que generaron más ventas en un análisis determinado.

Campo Tipo de Dato Permite Nulo Propósito / Justificación
ID_EmpleadoMasVentas INT No Clave primaria de la tabla.
ID_Analisis INT No Referencia al análisis correspondiente.
ID_Empleado INT No Identificador del empleado.
TotalGenerado DECIMAL(10,2) No Total de ventas generadas por el empleado.
