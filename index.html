Proyecto Integrador - Monkey Café

Resumen

Este proyecto consiste en el desarrollo de un sistema para la gestión de una cafetería, integrando módulos de Analisis de ventas, Marketing, Personal, Ventas, Ingredientes, Proveedores, Clientes, Inventario, Facturacion, Carta, Seguridad, Membresia.

Problemática

Actualmente, la cafetería no cuenta con un sistema que administre sus procesos de manera integral, lo que genera problemas de control en inventarios, ventas y reportes.

Objetivo

Desarrollar un sistema integral que permita gestionar de manera eficiente los productos, ingredientes, proveedores y ventas, implementando procesos automatizados y controlados que optimicen la operación diaria del negocio, faciliten la toma de decisiones y garanticen un control preciso de inventarios y recursos, con el fin de mejorar la productividad, la rentabilidad y la satisfacción del cliente.
