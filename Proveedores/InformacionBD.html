Documentación de la Base de Datos de Proveedores

Módulo: Proveedores

Diagrama de la Base de Datos

Tabla: Direccion

Propósito: Almacenar la información de ubicación de cada proveedor de manera estructurada.

Campo Tipo de Dato Permite Nulo Propósito / Justificación
ID_Direccion INT No Clave primaria que identifica de manera única cada dirección.
Calle NVARCHAR(200) Sí Calle donde se encuentra el proveedor.
Colonia NVARCHAR(100) Sí Colonia o barrio del proveedor.
CP NVARCHAR(10) Sí Código postal del proveedor.
Ciudad NVARCHAR(100) Sí Ciudad donde se encuentra el proveedor.
Estado NVARCHAR(100) Sí Estado o región del proveedor.
Pais NVARCHAR(50) Sí País donde se encuentra el proveedor.

Tabla: Proveedor

Propósito: Almacenar la información principal de cada proveedor y vincularlo con su dirección.

Campo Tipo de Dato Permite Nulo Propósito / Justificación
ID_Proveedor INT No Clave primaria que identifica de manera única cada proveedor.
Nombre NVARCHAR(100) No Primer nombre del proveedor.
SegundoNombre NVARCHAR(100) Sí Segundo nombre del proveedor, si aplica.
ApellidoP NVARCHAR(100) No Apellido paterno del proveedor.
ApellidoM NVARCHAR(100) Sí Apellido materno del proveedor, si aplica.
Telefono NVARCHAR(20) No Número de contacto del proveedor.
Correo NVARCHAR(100) Sí Correo electrónico del proveedor.
RFC_NIT NVARCHAR(20) No Identificación fiscal del proveedor.
Estatus NVARCHAR(10) No Indica si el proveedor está activo o inactivo.
FechaRegistro DATE No Fecha en que se registró el proveedor.
ID_Direccion INT No Llave foránea que enlaza el proveedor con su dirección en la tabla Direccion.

Tabla: ProveedorIngrediente

Propósito: Registrar qué ingredientes proporciona cada proveedor y en qué cantidad.

Campo Tipo de Dato Permite Nulo Propósito / Justificación
ID_ProveedorIngrediente INT No Clave primaria que identifica cada registro de relación proveedor-ingrediente.
ID_Proveedor INT No Llave foránea hacia la tabla Proveedor, identifica el proveedor que surte el ingrediente.
ID_Ingrediente INT No Identificador del ingrediente suministrado.
Cantidad DECIMAL(10,2) No Cantidad de ingredientes que proporciona el proveedor.
